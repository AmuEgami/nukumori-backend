# nukumori-backend
Spring Boot で構築した社内交流アプリのバックエンド。投稿・プロフィール管理・認証連携機能などを実装しています。

## 概要
- Spring Boot を使用したREST API構成
- AWS（DynamoDB / Cognito）と連携
- ユーザープロフィール登録、投稿管理を担当

## 技術スタック
- Java / Spring Boot
- AWS DynamoDB / Cognito
- Maven
